Hipótesis

Longitud rodilla-talón

Se quiere determinar si existe una diferencia significativa entre las mediciones realizadas con cinta métrica y con antropómetro medidas en los mismos individuos:

\[ \begin{cases} H_0: \, \, \, \mu_{CM} - \mu_{A} = 0 \\ H_1: \, \, \, \mu_{CM} - \mu_{A} \neq 0 \end{cases}\]

Donde \(\mu_{CM}\) representa la media de las mediciones realizadas con cinta métrica y \(\mu_{A}\) representa la media de las mediciones realizadas con antropómetro.

Inicialmente se hace una comparación gráfica de ambas técnicas a través de un box-plot:

Según el gráfico de cajas (boxplot), parece que existe diferencia entre las mediciones de la longitud desde la rodilla hasta el talón realizadas con cinta métrica y con antropómetro. Se evidencia que las mediciones realizadas con cinta métrica son en promedio superiores a las realizadas con antropómetro. Sin embargo, es necesario hacer una prueba estadística más formal. Para esto, se calculan las diferencias entre las mediciones de ambos métodos y se verifica si esas diferencias siguen una distribución normal para determinar qué prueba estadística es adecuada para la comparación.

A continuación se presentan los resultados de los test de Shapiro-Wilk y de Lilliefors para la diferencia de las mediciones LRT_CM y LRT_A:


    Shapiro-Wilk normality test

data:  LRT_dif
W = 0.89825, p-value < 2.2e-16

    Lilliefors (Kolmogorov-Smirnov) normality test

data:  LRT_dif
D = 0.1146, p-value < 2.2e-16

Los resultados de dos pruebas estadísticas, Shapiro-Wilk y Lilliefors, mostraron un p-valor menor a \(0.05\). Esto significa que, con un nivel de confianza del \(95\%\), no podemos concluir que las diferencias entre las mediciones de rodilla - talón tomadas con una cinta métrica y con un antropómetro sigan una distribución normal, y por lo tanto no es posible usar la prueba t para muestras pareadas. Por lo tanto, para analizar las diferencias en los promedios, es necesario usar un tipo de prueba que no dependa de suposiciones sobre la distribución de los datos.

El siguiente paso es evaluar el sistema de hipótesis mediante una prueba no paramétrica: Test de Rangos con Signo de Wilcoxon. Las hipótesis en este caso no son las mismas expuestas anteriormente. El sistema de hipótesis en este caso es el siguiente:

\[ \begin{cases} H_0: \textrm{La distribución de las diferencias es simétrica alrededor de 0}\\ H_1: \textrm{La distribución de las diferencias no es simétrica alrededor de 0} \end{cases}\]

La cuál es equivalente al siguiente sistema de hipótesis bajo \(H_0\):

\[ \begin{cases} H_0: \textrm{La mediana de las diferencias es 0}\\ H_1: \textrm{La mediana de las diferencias no es 0} \end{cases}\]

A continuación se presenta el resultado del test:


    Wilcoxon signed rank test with continuity correction

data:  datos$LRT_CM and datos$LRT_A
V = 144691, p-value < 2.2e-16
alternative hypothesis: true location shift is not equal to 0
95 percent confidence interval:
 1.425063 1.525013
sample estimates:
(pseudo)median 
      1.475033

El resultado del Test de Rangos con Signo de Wilcoxon mostró un p-valor de \(2.2 \times 10^{-16}\), lo que indica que en este caso con una confianza del \(95\%\) se rechaza la hipótesis nula de que la distribición de las diferencias sea simétrica alrededor de 0. Se concluye que las mediciones de la longitud rodilla - talón hechas con la cinta métrica son significativamente diferentes a las obtenidas con el antropómetro.
